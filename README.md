# Vue 3 + Vite + Motion Tutorial

本專案使用 Vite 建立 Vue3，並已安裝 motion-v 套件。

## 線上展示

🚀 **Live Demo**: [https://kanboo.github.io/motion-tutorial/](https://kanboo.github.io/motion-tutorial/)

## 快速開始

1. 安裝相依套件：
   ```sh
   npm install
   ```
2. 執行開發伺服器：
   ```sh
   npm run dev
   ```
3. 建構專案：
   ```sh
   npm run build
   ```
4. 預覽建構結果：
   ```sh
   npm run preview
   ```

## 已安裝套件
- motion-v
- tailwindcss
- element-plus
