# Vue 3 + Vite

本專案使用 Vite 建立 Vue3，並已安裝 motion-v 套件。

## 快速開始

1. 安裝相依套件：
   ```sh
   npm install
   ```
2. 執行開發伺服器：
   ```sh
   npm run dev
   ```

## 已安裝套件
- motion-v
